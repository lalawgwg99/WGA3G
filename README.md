## ✨ 功能說明

### 📷 智慧識別

- 🏪 店點名稱、📅 日期時間

### 🌟 圖片美化增強

上傳後自動處理：

- ✨ 銳化 (Sharpen) - 文字更清晰

### � 分享格式

1. **文字訊息**（含高清連結）
2. **原始照片**（附圖）

## 📸 使用流程

1️⃣ 上傳訂單照片  
2️⃣ 自動顯示：原圖 + 美化圖  
3️⃣ 點擊「開始識別」  
4️⃣ 檢查並修正識別結果  
5️⃣ 點擊「確認資料正確」分享

## 🔒 隱私說明

- AI 識別在伺服器端完成
- API Key 安全儲存於環境變數

## 🚀 部署設定

1. Fork 此倉庫並連接 Cloudflare Pages
2. 設定環境變數 `GEMINI_API_KEY`
3. 綁定 R2 Bucket：變數名 `IMAGES`
4. Build command 留空，Output directory 設為 `/`

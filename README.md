# 📱 訂單識別助手

> 🚀 Powered by **Gemini 2.0 Flash** AI

## ✨ 功能說明

上傳訂單圖片，AI 自動識別：

- 🏪 店別名稱（例如：XX店）
- 📅 日期時間
- 📋 訂貨編號 / 發票號碼
- 📦 商品代碼與品名（支援多商品）

**特色功能**：

- 識別結果可點擊編輯修正
- 確認後自動產生分享文字
- 清爽藍綠系介面設計

## 📸 使用流程

1️⃣ 上傳訂單照片  
2️⃣ 點擊「開始識別」  
3️⃣ 檢查並修正識別結果  
4️⃣ 點擊「確認資料正確」分享

## 🛠️ 技術架構

| 層級 | 技術 |
|------|------|
| 前端 | HTML + JavaScript (原生) |
| 後端 | Cloudflare Pages Functions |
| AI 模型 | Google Gemini 2.0 Flash |
| 安全 | API Key 存放於環境變數 |

## 🔒 隱私說明

- 所有圖片處理在雲端完成後即刪除
- 不儲存任何訂單資料
- API Key 安全儲存於伺服器端

## 🚀 自行部署

1. Fork 此倉庫
2. 連接至 Cloudflare Pages
3. 設定環境變數 `GEMINI_API_KEY`
4. 完成部署
